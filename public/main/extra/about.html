About TaskBerry

TaskBerry is a lightweight to-do app built to keep you moving. It favors a clean table layout, quick actions, and a calm flow—add a task, review at a glance, and check things off without friction.

About the developer

I’m Sonal “Sunny” Kamble, a Computer Science grad student at SUNY New Paltz. TaskBerry was my first project as a grad student—started in my first semester and leveled up over summer break. I enjoy building practical tools with readable code, clear UX, and solid fundamentals.

Why I built this

TaskBerry began as part of my Web Development Project (WDP) coursework. I wanted a real, end-to-end app: user accounts, routes, data, and a UI that stays out of your way. The result is simple on purpose—and easy to extend.

What you can do today
  • Register and log in
  • Create jobs (tasks) with descriptions
  • See tasks in a tidy table
  • Delete tasks you don’t need
What’s next
  • Inline edit for faster changes
  • Priorities, due dates & filters
  • Mobile polish & accessibility
  • Security hardening
Under the hood
  • HTML, CSS, Vanilla JS
  • Node.js + Express routes
  • Simple RESTful APIs
  • Clean, extendable structure

Thanks for visiting TaskBerry. I’m actively improving it—your feedback is welcome!
